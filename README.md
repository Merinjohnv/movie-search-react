#  Movie Search App (React)

A responsive movie search web application built using React and OMDb API.

##  Features
- Search movies in real-time
- Display movie title, poster, and year
- Loading indicator while fetching data
- "No movies found" handling
- Clear input button inside textbox
- Search using Enter key
- Responsive grid layout

##  Technologies Used
- React.js
- JavaScript (ES6)
- HTML5 & CSS3
- OMDb API
